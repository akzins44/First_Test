# Blueprint: Lotto Number Generator

## Overview

This document outlines the plan for creating a simple web-based lottery number generator. The application will generate and display 6 unique random numbers between 1 and 45.

## Current State

The project is a basic "Hello, world!" application with `index.html`, `main.js`, and `style.css` files.

## Plan

1.  **Modify `index.html`:**
    *   Add a title for the lottery number generator.
    *   Create a container to display the generated numbers.
    *   Add a button to trigger the number generation.
2.  **Modify `main.js`:**
    *   Implement a function to generate 6 unique random numbers.
    *   Add an event listener to the button to call the generator function and update the display.
3.  **Modify `style.css`:**
    *   Add styling for the numbers container, individual numbers, and the button to improve the visual presentation.
